## 二、视觉规格

### 2.1 背景
- body: `background: #F9FBFC;` 奶白色
- 无多余装饰，简洁干净
- canvas + CSS2D层叠（CSS2D在上层，用来放卡片）

### 2.2 点阵地球

**球体基础：**
- 半径 4.5，位置原点
- 内球：`SphereGeometry(4.35, 64, 64)`，材质 `MeshBasicMaterial`，color #EBF8FF，transparent=true，opacity=0.15
- 点阵球：用 `BufferGeometry` + `Points` 实现

**点阵大陆算法（简化但有效）：**
不用复杂的经纬度→陆地判断。改用以下方案：

1. 用斐波那契螺旋在球面上生成1800个均匀分布的点
2. 预定义7个大陆"矩形区域"（经纬度范围）：
```js
var continents = [
  { name:'北美', lat:[15,72], lng:[-170,-50] },
  { name:'南美', lat:[-55,12], lng:[-82,-34] },
  { name:'欧洲', lat:[36,71], lng:[-10,40] },
  { name:'亚洲', lat:[10,75], lng:[40,180] },
  { name:'非洲', lat:[-35,37], lng:[-17,52] },
  { name:'澳洲', lat:[-40,-10], lng:[113,155] },
  { name:'南极洲', lat:[-90,-65], lng:[-180,180] }
];
```
3. 判断每个斐波那契点的(lat, lng)是否落在任一矩形内
4. 在矩形内 = 陆地，保留该点；不在 = 海洋，丢弃
5. 最终陆地粒子约800-1000个

**点阵材质：**
- `PointsMaterial`，color #4A90E2，size 0.08，transparent=true，opacity=0.7
- 粒子圆形：用Canvas生成径向渐变sprite纹理（白色中心→透明边缘，32×32px）
- blending: NormalBlending（奶白背景下AdditiveBlending不可见）

**地球动画：**
- `earthGroup.rotation.y += 0.12 * deltaTime` 缓慢自转
- `earthGroup.rotation.x = 0.1` 轻微倾斜（固定值）
- 点阵和半透明内球放在同一个group里一起旋转

### 2.3 城市亮点 + 数据流

**城市亮点（8个）：**
从陆地粒子中随机选8个作为"城市"。在选中的粒子位置放置小球体：
- `SphereGeometry(0.12, 16, 16)`，`MeshBasicMaterial`，color #2B6CB0
- 外层光晕：`SphereGeometry(0.2, 16, 16)`，同色，opacity 0.15
- 每个城市点存储在 `cityPositions` 数组中（使用Vector3，放在earthGroup的local空间）

**数据流连线：**
每个城市点到球心(0,0,0)有一条连线：
- 使用 `Line` + `BufferGeometry`，两个顶点：cityPos → (0,0,0)
- `LineBasicMaterial`，color #BEE3F8，transparent=true
- 初始 opacity 0.25，4秒后渐变至 0.7（在render loop中判断time）
- 连线也要随地球旋转（放在earthGroup中）

**流动粒子（每条连线3个）：**
- 每个城市3个粒子小球（`SphereGeometry(0.05, 8, 8)`，color #3182CE）
- 粒子从cityPos沿连线向球心移动
- 移动进度：`((time * speed + offset) % 1)`，offset为0, 0.33, 0.66（三粒子均匀分布）
- 到达球心（进度=1）后重置回cityPos（进度=0）
- 4秒后 speed 从1.0逐渐增加到2.5
- 粒子也放在earthGroup中随地球旋转

**动画三阶段（用统一的elapsedTime控制）：**

阶段1（0-2.5s，汇聚）：
- 卡片初始位置：在半径18.75的球面上均匀分布（斐波那契7点）
- 卡片target位置：在半径7.5的球面上均匀分布
- 使用easeOutCubic插值：`t = elapsedTime / 2.5; r = 18.75 + (7.5-18.75) * ease(t)`
- 透明度：opacity从0线性增长到1
- 每个卡片在球面上的角度（theta, phi）保持不变，只有半径变

阶段2（2.5-8.0s，环绕 + 浮动）：
- 每个卡片围绕地球Y轴公转：`theta += 1.2 * deltaTime`
- 上下浮动：`Y += 0.3 * Math.sin(elapsedTime * 1.5 + cardIndex * 0.9)`
- 半径保持7.5
- 透明度保持1

阶段3（8.0s后，散开）：
- 半径从7.5线性增加到16.5（2秒内）
- 透明度从1线性降到0（2秒内）
- 半径达到16.5且opacity归零后，卡片隐藏（display:none或remove）

## 四、关键实现提示

### 4.1 斐波那契球面分布
```js
// 生成均匀分布在球面上的N个点
var points = [];
var phi = Math.PI * (3 - Math.sqrt(5));
for (var i = 0; i < N; i++) {
  var y = 1 - (i / (N - 1)) * 2;
  var radius = Math.sqrt(1 - y * y);
  var theta = phi * i;
  var x = Math.cos(theta) * radius;
  var z = Math.sin(theta) * radius;
  // (x,y,z) 是单位球面上的点，再乘以地球半径
  // 转为经纬度：
  var lat = Math.asin(y) * 180 / Math.PI;
  var lng = Math.atan2(z, x) * 180 / Math.PI;
  // 判断是否在任一大陆矩形内
}
```

### 4.2 三阶段时间管理
```js
var elapsedTime = 0;
function animate(timestamp) {
  requestAnimationFrame(animate);
  if (!lastTime) lastTime = timestamp;
  var dt = Math.min((timestamp - lastTime) / 1000, 0.1);
  lastTime = timestamp;
  elapsedTime += dt;

  // 阶段判断
  var phase = elapsedTime < 2.5 ? 'gather' : elapsedTime < 8.0 ? 'orbit' : 'scatter';
  
  updateCards(elapsedTime, phase);
  updateEarth(dt);
  updateCityParticles(elapsedTime, dt);
  // ...
}
```

### 4.3 CSS2D卡片更新
```js
function updateCards(t, phase) {
  for (var i = 0; i < cards.length; i++) {
    var card = cards[i];
    var baseAngle = (i / 7) * Math.PI * 2; // 均匀分布
    var radius;
    if (phase === 'gather') {
      var progress = Math.min(t / 2.5, 1);
      var eased = 1 - Math.pow(1 - progress, 3); // easeOutCubic
      radius = 18.75 + (7.5 - 18.75) * eased;
      card.object.element.style.opacity = progress;
    } else if (phase === 'orbit') {
      radius = 7.5;
      card.object.element.style.opacity = 1;
    } else {
      var scatterProgress = Math.min((t - 8.0) / 2.0, 1);
      radius = 7.5 + (16.5 - 7.5) * scatterProgress;
      card.object.element.style.opacity = 1 - scatterProgress;
    }
    var theta = baseAngle + (phase === 'orbit' ? t * 1.2 : 0);
    var yOffset = (phase === 'orbit') ? 0.3 * Math.sin(t * 1.5 + i * 0.9) : 0;
    card.object.position.set(
      Math.cos(theta) * radius,
      yOffset,
      Math.sin(theta) * radius
    );
  }
}
```
