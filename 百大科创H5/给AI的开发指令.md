# 给 DeepSeek V4 Pro 的开发指令清单

## 项目背景（每次新对话先贴这段）

```
我在开发一个参赛H5科普作品——"AI奇妙之旅：从魔法到科学"。
纯HTML+CSS+JS，零框架，桌面端优先（1200px）。
项目结构：
  index.html  ← 主页面（首页+5站）
  css/style.css ← 样式
  js/data.js   ← 题库/时间轴/对话数据
  js/main.js   ← 交互逻辑

已有功能：时间轴卡片、数据喂养模拟器、决策树、偏见实验室、
AI实验室（5个实验台模拟回复）、闯关答题（5关+徽章+前后测）。

目标用户：10-15岁青少年。给代码时直接给完整文件内容。
```

---

## 指令一：接入真实AI API（最重要）

```
当前AI实验室的5个实验台用的是模拟回复（setTimeout假数据）。
请帮我改成接入deepseek-v4-proAPI：sk-c037224e1f1544fcbf1e459e1771a8f4
1. 在js/下新建 api.js，封装API调用函数
2. API Key 用变量存储（方便我替换）
3. 5个实验台（问答/写诗/识别/游戏工坊/生活助手）用不同的system prompt
4. 加上loading状态和错误处理
5. 修改main.js中submitLab函数，替换模拟回复为真实API
```

## 指令二：增加神经网络动画

```
第二站实验3目前只有静态的CSS圆点。
请用Canvas画一个真正的神经网络动画：
- 3层结构（输入层3节点→隐藏层4节点→输出层2节点）
- 数据以光点形式从左向右脉冲流动
- 点击"输入图片"按钮触发一次流动动画
- 纯JS实现，无需任何库
```

## 指令三：数据喂养模拟器加拖拽

```
第二站实验1目前是点击触发，不够直观。
```
